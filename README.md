# My Todo App

A full-stack web application for managing tasks and collaborating with team members.

This project was built as a personal practice project to learn full-stack development, including backend APIs, database integration, and cloud deployment.

## Live Demo
https://my-todo-app-fdur.onrender.com

You can explore the application **without creating an account** using **Demo Mode**.

---

## Technologies

**Frontend**
- HTML
- CSS
- JavaScript
- Chart.js

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB Atlas
- Mongoose

**Deployment**
- Render

**Version Control**
- Git
- GitHub

## Features

### Demo Mode
Users can explore the system without logging in.

- Uses demo data
- Does not affect real user data
- A banner shows whether the system is in **Demo Mode** or **Real Mode**

---

### Task Management
Users can:

- Create tasks
- Edit tasks
- Set priority
- Set due dates
- Assign categories
- Tag other members

---

### Categories
Users can create task categories with:

- Name
- Icon
- Color

---

### Dashboard
The dashboard shows:

- Total tasks
- Pending tasks
- Tasks in progress
- Completed tasks
- Upcoming tasks
- Overdue tasks
- Task statistics charts
